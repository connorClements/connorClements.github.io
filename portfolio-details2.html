Project Details

Intended as a mobile-first application, the 'Company Directory' can be used to keep track of the different departments, personal details and locations of all it's employees. Users can add new locations, departments and employees, update employee details and delete departments, locations and employees if necessary. All of the details are displayed in a table format and is updated in real-time. The application makes use of jQuery, SQL and PHP to create, read, update and delete all of the information in the Company Directory, and the project is styled using Bootstrap.
